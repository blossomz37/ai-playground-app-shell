# Agent Workspace

This directory holds agent-facing project memory for this repo. It is intentionally small and should point to canonical project docs rather than replace them.

## Directories

- `knowledge/` - stable orientation that should save future agents from re-researching the workspace.
- `notes/` - lightweight session observations that are useful but not authoritative.
- `protocols/` - short standing workflows agents should follow for recurring repo operations.

## Canonical Sources

- Root agent guide: `../AGENTS.md`
- Durable project guide: `../CLAUDE.md`
- Latest handoff: `../workspace-agents/session-handoffs/HANDOFF_NN.md`
- Implementation conventions: `../workspace-agents/implementation/AGENTS.md`
- Repo maintenance protocol: `protocols/REPO_MAINTENANCE.md`

If this directory drifts from the project docs, update the stale file and keep the canonical source clear.
