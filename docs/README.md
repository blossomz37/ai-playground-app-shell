# Project Documentation

This folder contains shared developer-facing documentation for App Shell.

## Start Here

- `architecture/shell-platform-spec.md` - platform decisions, primitives, and resolved architecture choices.
- `architecture/shell-spec.md` - shell implementation/spec detail.
- `architecture/module-contract.md` - module boundary and extension API.
- `architecture/modules-overview.md` - first-party module map and ownership split.
- `docs/modules/` - module-specific design specs.
- `docs/reference/` - prior art, anchor analysis, and reusable reference material.
- `docs/product/` - product-facing requirements when they are split out from architecture.

Agent planning, handoffs, and screenshot evidence live in `../workspace-agents/`, not here.
